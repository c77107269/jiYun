1.前端编程的概述
  HTML、CSS、JavaScript是前端开发的三大核心
  HTML和CSS是静态前端的基础
  JavaScript是动态前端的应用开发的核心；从交互的角度来看，是提升用户体验前端技术的标准
  
2.前端页面的组成
  分成：结构层、表示层、行为层
  
  结构层（structural layer）
    由HTML或XHTML之类的标记语言负责创建标签，对网页内容的语以含义做出描述，但这些标签不包含任何关于如何显示有关内容的信息。
    
  表示层（presentation layer）
    由CSS负责创建。CSS定义了“如何显示有关内容”。
  
  行为层（behavior layer）
    负责“内容如何对事件做出反应”这一问题。这个是JavaScript语言和DOM主宰的领域。
  
  结构：html，从语义的角度，描述页面结构
  样式：css，从审美的角度，美化界面样式
  行为：JavaScript，从交互的角度，提升用户体验前端技术标准；是动态前端开发的应用核心

  2.JavaScript概述
    JavaScript是Web开发人员必须学习和掌握的3门语言中的一门；
    HTML 定义了网页的内容；
    CSS 描述了网页的布局；
    JavaScript 网页的行为

  js组成
    1、ECMAScript(3/5/6/7)它是JS语言的标准,规定了JS的编程语法和基础核心知识
    2、DOM: document object model 文档对象模型,提供给JS很多操作页面中元素的属性和方法,可以删除、添加或者替换节点
    3、BOM: browser object model 浏览器对象模型 ,提供了很多操作浏览器的属性方法,而这些方法都存放在window浏览器对象上
