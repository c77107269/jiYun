1、js变量的声明（var/let/const)
	js的变量声明方式有两种：
		a> 声明和赋值在同一行
			var str = '声明一个字符串'
		b> 先声明，再赋值
			var str1
			str1 = '声明的第二个字符串'
2、在JS中所有的我们可以自主命名的都可以称之为 标识符。例如：变量名、函数名、属性名都属于标识符。
3、命名标识符的规则：
	1、标识符中可以含有字母，数字，下划线 _，$		var a_1 = '字符串'
	2、标识符不能以数字开头。用字母，下划线 _，$开头都没问题		var 3 = '字符串'
	3、标识符不能是ES中的关键字或是保留字		var var = ''
	4、标识符一般都采用驼峰命名法。首字母小写，每个单词的开头字母大写，其余字母小写。
