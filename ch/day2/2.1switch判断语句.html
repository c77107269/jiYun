1、switch结构
        switch(变量)
        {
            case 值0:
            case 值1:
            {//当此处内容较少时括号可以不加
                变量 === 值1   时执行的语句;
                break;
            }
            case 值2:
            {
                变量 === 值2   时执行的语句;
                break;
            }
            case 值3:
            {
                变量 === 值3   时执行的语句;
                break;
            }
            default:
            {
                变量不等于以上任何一个值时执行的语句;
            }
        }
        其中break 用于结束当前分支，否则将一直向下执行，直到遇到下一个 break语句才跳出。

        某些情况，可以不同的条件执行相同的结果
        switch(x){
            case 1:
            case 2:
            case 3:
        }




    2、switch()括号内的语句，和case：冒号后面的代码块部分，都可以使用表达式，
        但是需要注意的是两者后面的表达式在运行结果时，采用的是'==='，也就是说，在比较时，不会发生类型的转换
